# SetuAI 🇮🇳 – AI for Bharat

SetuAI is a multilingual AI-powered assistant designed to help rural citizens discover eligible government schemes based on income, occupation, and category.

## 🚀 Features

- Multilingual support (English, Hindi, Marathi, Tamil, Bengali)
- AI explanation using Amazon Bedrock (Claude 3 Haiku)
- Low bandwidth mode
- Voice interaction support
- Profile-based eligibility filtering
- Deployed on Render + Vercel

## 🏗 Architecture

Frontend: React (Vercel)
Backend: Node.js + Express (Render)
AI Layer: Amazon Bedrock (Claude 3 Haiku)
Cloud: AWS

## 🎯 Problem Solved

Bridges information gap for rural citizens by simplifying complex government schemes into local languages using AI.
